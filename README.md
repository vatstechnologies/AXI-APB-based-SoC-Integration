# Mini AXI/APB SoC with Cache

## Features
- Simplified AXI-lite CPU
- APB bridge
- Direct-mapped cache
- Memory-mapped architecture

## How to Run
- Go to EDA Playground
- Select Icarus Verilog
- Run simulation

## Learning Outcomes
- SoC architecture
- Cache design basics
- Bus protocols (AXI/APB)
